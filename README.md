# ABill Invoice Generator for WooCommerce

Professional PDF Invoice Generator for WooCommerce.

ABill Invoice Generator helps WooCommerce store owners create, manage, print, download and email professional invoices directly from the WordPress dashboard.

## Features

- Create professional invoices
- Automatic invoice numbering
- Edit invoice details
- Print invoices
- Download invoices as PDF *(coming soon)*
- Email invoices to customers *(coming soon)*
- WooCommerce integration
- Business information settings
- Clean and lightweight
- Developer friendly
- Translation ready

## Requirements

- WordPress 6.8 or later
- WooCommerce 9.0 or later
- PHP 8.0 or later

## Installation

1. Upload the plugin to `/wp-content/plugins/`
2. Activate the plugin.
3. Go to **ABill → Settings**.
4. Configure your business information.
5. Start creating invoices.
